# calendar-electron

## 一个做着玩的electron集成vue项目
## 分为学习复习页面和面试页面

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### start electron
```
npm run start
```

### package
```
npm run package
```
